How to Start a Django Project
on Windows 11

Get started with Django on Windows 11

Friday, 9th of June, 2023

If you've been curious about getting your feet wet in web development, Django could be the perfect place to start. Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design.


Whether you're a coding novice or a seasoned veteran, setting up a Django project on your Windows 11 machine can be tricky because of you OS, or the version you are trying to install. In this blog post, we'll walk you through the process step-by-step.


Introduction:


First and foremost, even with a YouTube video, creating a Django project can be challenging, particularly if you are new to virtual environments or web development.


Not to mention that each update brings slight changes, and I can't fault YouTubers for not producing four videos a year on how to set up Django in Python.


Struggling to follow a tutorial on how to create a web app, then spending nearly an hour trying to figure out how to install the "pip" package or the software can lead to demotivation.


It can diminish your enthusiasm and disrupt your momentum. So, what could be better than a five-step tutorial with clear commands that you can either type into your "PowerShell" or simply copy and paste if you're a "Ctrl+C / Ctrl+V" professional?


Step 1: Setting Up Your Directory


Before starting your project, you'll need to identify a directory where you'll host it.
You can choose any location on your computer that suits you, as long as you remember where it's located. To navigate to your chosen directory, you'll need to use either PowerShell or the Command Prompt. Type in cd, followed by the path of your directory.
For instance, if your folder is located in the Documents folder and is called 'MyProject', you would type:

C:\User\myPC> cd Documents\MyProject


Step 2: Setting Up the Virtual Environment


Pythonâ€™s virtual environments allow you to isolate the Python interpreter, libraries, and scripts in a specific directory.
This means that every project can have its own dependencies, regardless of what dependencies every other project has.
To create a virtual environment for your Django project, enter in your Command Prompt:

C:\User\myPC\Documents\MyProject> py -m venv your_project_name


Step 3: Activating the Virtual Environment


Now that you've set up your virtual environment, you need to activate it.
Activation changes the behavior of your shell session, so that Python can find and use the packages installed in the virtual environment.
Enter :

C:\User\myPC\Documents\MyProject> your_project_name\Scripts\activate.bat

to activate the environment.

After activation, navigate to your project's directory by typing:

C:\User\myPC\Documents\MyProject> cd your_project_name


Step 4: Setting Up Application Files and Manage.py File


Now it's time to create your Django project's core files.
Run:

C:\User\myPC\Documents\MyProject> python -m django startproject your_project_name

This command will create a new Django project in your current directory.
Once the setup is complete, confirm that everything is properly installed by typing:

C:\User\myPC\Documents\MyProject> python -m django

or

C:\User\myPC\Documents\MyProject> python manage.py

If there's no error, you've successfully installed Django! To begin coding, open your preferred text editor or IDE. For instance, if you use Visual Studio Code, just enter:

C:\User\myPC\Documents\MyProject> code .

to open the current directory in the editor.


Step 5: Running the Server


With your Django project set up, you can now run the server to test your application.
Enter:

C:\User\myPC\Documents\MyProject>python manage.py runserver

to start the server.

By default, the server runs on port 8000, but if you need to specify a different port, you can do so by adding it after runserver
e.g.:

C:\User\myPC\Documents\MyProject>python manage.py runserver 7000

Once the server is running, you can access your site by clicking the provided:
- http://link:8000/ (or whatever port you've specified).
If you're running in PowerShell or CMD, you can simply hold Ctrl and click the link to open it in your default web browser.
And here we are! Be proud of yourself, you've successfully set up your first Django project on Windows 11.
Happy coding!
